# PQB Website - Polska Rada Jakości

Website for the Polish Quality Board (PQB) - a national organization dedicated to software testing certification and quality assurance.

**Live Site:** [https://pqb.org.pl](https://pqb.org.pl)

## About PQB

Polska Rada Jakości (Polish Quality Board) is an authorized ISTQB partner in Poland that provides:

- Professional certifications (ISTQB)
- Training and workshops
- Quality assurance expertise
- International collaboration in IT quality standards
